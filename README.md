# creative-media-finder
A functional, human-centric Python application utilizing structured databases to catalog, search, and aggregate user evaluations for cinema and musical works. This project demonstrates backend algorithmic design, input data parsing, and dynamic numeric adjustments.

##Core Features
* **Cinematic Database**: Built-in repository tracking artistic themes, genres, and ratings.
* **Algorithmic Search**: Case-insensitive search optimization for seamless user exploration.
* **Interactive Evaluation**: Live runtime terminal inputs that recalculate dynamic averages using aggregate mathematical arrays.

 ###Tech Stack & Concepts
* **Language**: Python 3.x
* **Core Mechanisms**: Control Flow, Dictionaries, Function Definitions, Error Mitigation, Interactive Terminal Buffers.
